# 🏠 ML House Price Prediction

A full-stack Machine Learning project that predicts house prices based on user inputs like location, square feet, number of bedrooms (BHK), and bathrooms.

---

## 🚀 Features

- 📊 House price prediction using Machine Learning
- 🌐 Flask-based backend API
- 🎨 Simple frontend (HTML, CSS, JS)
- 📍 Location-based predictions
- ⚡ Fast and responsive

---

## 🧠 Tech Stack

- Python
- NumPy
- Pandas
- Scikit-learn
- Flask
- HTML, CSS, JavaScript
